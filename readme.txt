Landing Page, páginas web
Esta página es una página web, para un cliente que vende 
tiendas online. tanto el css como el javascript se ecuentran 
inline dentro del archivo landing.html.

Consta de las siguientes Secciones:

header
en esta sección encontraremos en el lado derecho el logo de la empresa
a la izquierda encontraremos el menú de navegacion.


slider
el slider cuenta con cuatro imágenes, con una velocidad de 
3000 para que sea una transición suave. 
la etiqueta del slider se encuentra en la línea 40.
y el codigo de javascript  a partir de la linea 100.


section1
En la sección1 construi una tabla para crear una sección donde 
a la izquierda está el texto y a la derecha está la imagen. 

footer
el footer es un div. con el mismo color que el div con el id header.
Ahi va la info de redes sociales y los datos de contacto.
